# Introduction

DB9S is a TUI(Terminal User Interface) for connecting to various types of databases.

The name is a rip-off of the lovely K9S TUI application.

Supported databases:
- [x] MySQL
- [x] PostgreSQL
- [x] Redis
- [x] SQLite
- [ ] ElasticSearch/Opensearch
- [ ] MongoDB
- [ ] Kafka
- [ ] NATS
- [ ] S3
- [ ] GCS
- [ ] TiKV
